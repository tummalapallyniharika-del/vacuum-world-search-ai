# Vacuum World AI Project

This project implements the Vacuum World problem using classical AI search algorithms.
It was developed as an academic assignment for an Artificial Intelligence course.

---

## Problem Description

A vacuum cleaner agent moves in a grid environment that contains dirty cells.
The goal of the agent is to clean all dirty cells using the minimum number of actions.

Actions available:
- Move Up
- Move Down
- Move Left
- Move Right
- Clean

Each action has a cost of 1.

---

## Algorithms Implemented

- Depth First Search (DFS)
- A* Search
- Iterative Deepening A* (IDA*)
